## Service Philosophy

Every customer interaction is an opportunity to earn loyalty. Our service standards are built on three commitments:

1. **Acknowledge** — Every customer is greeted within 30 seconds of entering the store or approaching an associate.
2. **Assist** — We listen first, recommend second, and never pressure.
3. **Appreciate** — Every transaction ends with a genuine thank-you and an invitation to return.

> **Standard**: If a customer can see you, you can greet them. If a customer needs help, everything else waits.

## Customer Engagement Model

### The SERVICE Framework

| Step                        | Action                                                          | Example                                                        |
| --------------------------- | --------------------------------------------------------------- | -------------------------------------------------------------- |
| **S** — Smile & greet       | Make eye contact, smile, offer a warm greeting                  | "Welcome in! Let me know if I can help you find anything."     |
| **E** — Engage              | Ask open-ended questions to understand needs                    | "What brings you in today?" / "Who are you shopping for?"      |
| **R** — Recommend           | Suggest products based on stated needs                          | "Based on what you described, I'd recommend..."                |
| **V** — Value-add           | Offer complementary items, loyalty program, or styling advice   | "This belt pairs perfectly with those trousers."               |
| **I** — Invite to purchase  | Guide to checkout when the customer is ready                    | "I can hold these at the register whenever you're ready."      |
| **C** — Close warmly        | Thank the customer; mention upcoming events or promotions       | "Thanks for coming in — we have new arrivals dropping Friday!" |
| **E** — Ensure satisfaction | Confirm the customer has everything they need before they leave | "Is there anything else I can help you with today?"            |

## Service Level Standards

### Response Time Targets

| Touchpoint            | Standard                                  | Measurement               |
| --------------------- | ----------------------------------------- | ------------------------- |
| Greeting upon entry   | ≤ 30 seconds                              | Mystery shop              |
| Fitting room check-in | ≤ 2 minutes after entering                | Mystery shop              |
| Checkout wait time    | ≤ 3 minutes (or open additional register) | POS queue timer           |
| Phone answer          | ≤ 3 rings (≤ 15 seconds)                  | Phone system report       |
| Email response        | ≤ 24 business hours                       | Ticketing system          |
| Social media response | ≤ 2 hours during business hours           | Social platform analytics |

### Queue Management

| Queue Length  | Action                                                                            |
| ------------- | --------------------------------------------------------------------------------- |
| 1–2 customers | Standard single-register operation                                                |
| 3 customers   | Open second register; page backup cashier                                         |
| 5+ customers  | All available associates to registers; MOD manages floor                          |
| 8+ customers  | Activate express checkout; apologize for wait; offer discount code for next visit |

## Complaint Handling

### LEARN Model

| Step              | Action                                                             | Key Phrases                                                    |
| ----------------- | ------------------------------------------------------------------ | -------------------------------------------------------------- |
| **L** — Listen    | Let the customer speak without interruption; take notes if complex | "I want to make sure I understand the full situation."         |
| **E** — Empathize | Acknowledge the frustration; validate their experience             | "I completely understand why that's frustrating."              |
| **A** — Apologize | Take ownership on behalf of the company; no blame-shifting         | "I'm sorry we let you down on this."                           |
| **R** — Resolve   | Offer a solution; give options where possible                      | "Here's what I can do for you..."                              |
| **N** — Notify    | Document the complaint; follow up if promised; escalate if needed  | "I've logged this and our team lead will follow up by [date]." |

### Escalation Matrix

| Issue Severity | Examples                                                          | First Responder           | Escalation                           |
| -------------- | ----------------------------------------------------------------- | ------------------------- | ------------------------------------ |
| **Low**        | Wrong size in bag, minor product question                         | Sales Associate           | —                                    |
| **Medium**     | Defective product, pricing error, rude service claim              | Shift Lead                | Store Manager if unresolved          |
| **High**       | Safety concern, discrimination allegation, viral social complaint | Store Manager (immediate) | District Manager + HR within 2 hours |
| **Critical**   | Legal threat, injury on premises, media inquiry                   | Store Manager (immediate) | VP Operations + Legal within 1 hour  |

### Complaint Documentation

All medium, high, and critical complaints are logged in the **Customer Feedback System (CFS)** with:

| Field                                   | Required |
| --------------------------------------- | -------- |
| Date and time                           | Yes      |
| Customer name and contact (if provided) | Yes      |
| Nature of complaint (category)          | Yes      |
| Description of incident                 | Yes      |
| Associate(s) involved                   | Yes      |
| Resolution offered                      | Yes      |
| Resolution accepted (Y/N)               | Yes      |
| Follow-up required (Y/N + date)         | Yes      |
| Logged by (associate name + ID)         | Yes      |

## Return & Exchange Policy

### Standard Return Policy

| Condition              | Timeframe             | Refund Method                         | Receipt Required                  |
| ---------------------- | --------------------- | ------------------------------------- | --------------------------------- |
| Unworn, tags attached  | 30 days from purchase | Original payment method               | Yes                               |
| Unworn, tags attached  | 30 days, no receipt   | Store credit at current selling price | No (ID required)                  |
| Unworn, tags attached  | 31–60 days            | Store credit only                     | Yes                               |
| Worn or altered        | Not accepted          | —                                     | —                                 |
| Defective merchandise  | 90 days               | Full refund or exchange               | Yes (waived with proof of defect) |
| Gift receipt           | 30 days               | Store credit at purchase price        | Gift receipt                      |
| Final sale / clearance | No returns            | —                                     | —                                 |

### Exceptions Requiring Manager Approval

| Scenario                                            | Authorization Level                          |
| --------------------------------------------------- | -------------------------------------------- |
| Return without receipt, value > $50                 | Shift Lead or above                          |
| Return outside standard window (31–60 days)         | Store Manager                                |
| Return of electronics or high-theft items           | Store Manager + LP review                    |
| Multiple returns from same customer in 30 days (>3) | Store Manager; flag in CFS                   |
| Price adjustment on previously purchased item       | Shift Lead (within 14 days of purchase only) |

### Return Processing Procedure

| Step | Action                                                                      |
| ---- | --------------------------------------------------------------------------- |
| 1    | Greet customer; ask for receipt or order confirmation                       |
| 2    | Inspect merchandise — verify condition, tags, and eligibility               |
| 3    | Scan receipt barcode or look up transaction by credit card / loyalty number |
| 4    | Process return in POS; select correct refund method                         |
| 5    | If store credit, print gift card and hand to customer                       |
| 6    | Return merchandise to restock queue (if sellable) or damage bin             |
| 7    | Thank customer and invite them to continue shopping                         |

## Loyalty Program Integration

### Program Overview

| Element         | Detail                                           |
| --------------- | ------------------------------------------------ |
| Program name    | [Company] Rewards                                |
| Enrollment      | Free; in-store, online, or via app               |
| Earn rate       | 1 point per $1 spent                             |
| Redemption      | 100 points = $5 reward                           |
| Bonus events    | 2x points during member-exclusive events         |
| Birthday reward | $10 off coupon, valid 30 days                    |
| Tier structure  | Base → Silver (500 pts/yr) → Gold (1,000 pts/yr) |

### Associate Responsibilities

- Ask **every** customer if they are a rewards member at checkout.
- If not enrolled, offer a brief explanation and enroll with consent.
- Target: **capture rate ≥ 70%** of transactions linked to a loyalty account.
- Never create accounts without customer knowledge and consent.

## Accessibility & Inclusivity Standards

| Standard                   | Requirement                                                                                                                  |
| -------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| **ADA / AODA compliance**  | Aisles minimum 36" wide; accessible fitting rooms available and unlocked                                                     |
| **Assistive service**      | Associates assist customers with mobility devices, retrieve items from high shelves, carry purchases to vehicle upon request |
| **Language**               | Bilingual signage in markets where required; translation app available on store tablet                                       |
| **Service animals**        | Welcomed without question; do not ask about the nature of the disability                                                     |
| **Sensory-friendly hours** | Designated quiet shopping hours (reduced lighting, no music) where offered                                                   |
| **Communication**          | Pen and paper available for customers who are deaf or hard of hearing; patience is mandatory                                 |

### Feedback Channels

| Channel                 | Method                                     | Frequency                   |
| ----------------------- | ------------------------------------------ | --------------------------- |
| Post-purchase survey    | Email / SMS triggered by POS transaction   | Every transaction (sampled) |

| Social media monitoring | Brand mentions, reviews, direct messages   | Continuous                  |
| Customer advisory panel | Quarterly focus group with loyal customers | Quarterly                   |

### Net Promoter Score (NPS)

```
NPS = % Promoters (9–10) − % Detractors (0–6)
```

| Score Range | Classification    |
| ----------- | ----------------- |
| 0–30        | Needs improvement |
| 31–50       | Good              |
| 51–70       | Excellent         |
| 71–100      | World class       |

**Company target NPS**: ≥ 50.
