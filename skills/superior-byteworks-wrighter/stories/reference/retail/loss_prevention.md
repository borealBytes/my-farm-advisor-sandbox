## Shrinkage Overview

Shrinkage — the difference between book inventory and physical inventory — is the single largest controllable expense in retail operations. This policy establishes the controls, procedures, and accountability framework to minimize shrinkage across all categories.

### Shrinkage Categories

| Category                         | Typical % of Total Shrink | Description                                          |
| -------------------------------- | ------------------------- | ---------------------------------------------------- |
| **External theft (shoplifting)** | 35–40%                    | Customer theft, organized retail crime (ORC)         |
| **Internal theft (employee)**    | 30–35%                    | Associate theft of merchandise, cash, or data        |
| **Administrative error**         | 15–20%                    | Pricing mistakes, receiving errors, mis-shipments    |
| **Vendor fraud**                 | 5–10%                     | Short shipments, billing discrepancies, return fraud |
| **Damage / waste**               | 3–5%                      | Breakage, spoilage, unsalvageable returns            |

**Company shrinkage target**: ≤ 1.5% of net sales.

## External Theft Prevention

### Deterrence Measures

| Control                                   | Standard                                                                                     | Responsible              |
| ----------------------------------------- | -------------------------------------------------------------------------------------------- | ------------------------ |
| **Customer acknowledgment**               | Greet every customer entering the store within 30 seconds                                    | All associates           |
| **Zone coverage**                         | Every sales floor zone staffed during operating hours; no blind zones                        | MOD                      |
| **Fitting room control**                  | Count items in / count items out; limit of 6 items at a time                                 | Fitting Room Associate   |
| **EAS (Electronic Article Surveillance)** | Hard tags on all merchandise > $20 retail; soft tags on remaining                            | Receiving / Floor Team   |
| **CCTV**                                  | Cameras covering entrances, exits, POS, fitting rooms (exterior), and high-theft zones       | LP / Facilities          |
| **Merchandise protection**                | Spider wraps on electronics; locked cases for high-value items; cable locks on display units | Visual Merchandiser / LP |
| **Receipt checks**                        | Bag check at exit for non-bagged or high-value items (where legally permitted)               | Door Greeter / Associate |

### Shoplifting Response Protocol

| Step | Action                                                                                                  | Who             |
| ---- | ------------------------------------------------------------------------------------------------------- | --------------- |
| 1    | **Observe** — Confirm suspicious behavior; do not confront based on appearance alone                    | Associate       |
| 2    | **Report** — Notify MOD or LP officer via radio using code language (e.g., "Code 10 in zone 3")         | Associate       |
| 3    | **Monitor** — LP / MOD maintains visual contact; documents on CCTV if possible                          | LP / MOD        |
| 4    | **Approach (if trained)** — Offer customer service: "Can I help you find anything?"                     | LP / MOD        |
| 5    | **Apprehension (LP only)** — If all five elements met (see below), trained LP may detain                | LP Officer only |
| 6    | **Document** — Complete Incident Report regardless of outcome                                           | LP / MOD        |
| 7    | **Law enforcement** — Contact police for all apprehensions; do not release without police report number | LP / MOD        |

### Five Elements Required for Apprehension

All five must be confirmed before any physical detention:

1. **Approach** — Associate observed the subject approach the merchandise.
2. **Selection** — Associate observed the subject select the merchandise.
3. **Concealment** — Associate observed the subject conceal or carry the merchandise in an unusual manner.
4. **Continuous observation** — Associate maintained unbroken observation of the subject.
5. **Exit** — Subject passed all points of sale and exited (or attempted to exit) the store without paying.

> **Critical**: If any element is broken, do not detain. Document and report to LP for investigation. Safety of associates and customers always takes priority over merchandise recovery.

## Internal Theft Prevention

### Controls

| Control                                | Description                                                                        | Frequency               |
| -------------------------------------- | ---------------------------------------------------------------------------------- | ----------------------- |
| **Background checks**                  | Pre-employment screening for all new hires                                         | At hire                 |
| **Cash handling separation of duties** | No single associate counts, reconciles, and deposits alone                         | Every shift             |
| **Register integrity**                 | Random register audits; no shared register drawers                                 | Weekly                  |
| **Void / no-sale monitoring**          | All voids, no-sales, and post-void transactions reviewed                           | Daily (automated alert) |
| **Refund monitoring**                  | Refunds > $50 require manager approval; pattern analysis on high-refund associates | Daily (automated alert) |
| **Discount authorization**             | Employee discount limited to self and immediate family; verified at POS            | Every transaction       |
| **Bag checks (associates)**            | Associates subject to bag/package inspection upon exit per employment agreement    | Every shift exit        |
| **Stockroom access**                   | Restricted to authorized personnel; access logged                                  | Continuous              |
| **Anonymous tip line**                 | Confidential hotline for reporting suspected internal theft                        | Continuous              |

### Red Flags — Internal Theft Indicators

| Indicator                                                 | Monitoring Method                         |
| --------------------------------------------------------- | ----------------------------------------- |
| High void rate relative to peers                          | POS exception report                      |
| Frequent post-void transactions                           | POS exception report                      |
| Register shortages on same associate's shifts             | Cash reconciliation                       |
| Refund activity with no corresponding customer present    | CCTV + POS cross-reference                |
| Unauthorized after-hours access                           | Alarm / access log review                 |
| Living beyond apparent means                              | Manager observation (discretion required) |
| Resistance to being observed or audited                   | Manager observation                       |
| Merchandise in associate-only areas (lockers, break room) | Periodic inspection                       |

### Investigation Procedure

| Step | Action                                                                     | Owner                   |
| ---- | -------------------------------------------------------------------------- | ----------------------- |
| 1    | LP receives report or system alert                                         | LP Analyst              |
| 2    | Preliminary review — POS data, CCTV, access logs                           | LP Analyst              |
| 3    | If evidence supports further investigation, open formal case               | LP Manager              |
| 4    | Covert investigation — additional monitoring, controlled transactions      | LP Team                 |
| 5    | Interview — conducted by LP Manager with HR present; documented in writing | LP Manager + HR         |
| 6    | Outcome — termination, prosecution, or cleared (per evidence)              | LP Manager + HR + Legal |
| 7    | Case file archived with all documentation                                  | LP Admin                |

> **Legal notice**: All investigations comply with applicable labor laws. Employees are presumed innocent until evidence supports a conclusion. Entrapment is prohibited.

## Administrative Error Reduction

### Common Error Types & Controls

| Error Type                         | Control                                                       | Owner                |
| ---------------------------------- | ------------------------------------------------------------- | -------------------- |
| **Receiving shortages not logged** | Mandatory ASN scan + discrepancy report                       | Receiving Lead       |
| **Pricing errors at POS**          | Weekly 50-SKU price audit                                     | Store Manager        |
| **Mis-ticketed merchandise**       | Spot-check during receiving (10% of cartons)                  | Receiving Lead       |
| **Transfer errors**                | Dual-scan at ship and receive; reconcile within 48 hrs        | Inventory Associate  |
| **Markdown errors**                | Automated markdown batches; manual overrides require approval | Pricing Ops          |
| **Damaged goods not written off**  | Daily damage bin processing; write-off in IMS same day        | Floor Lead           |
| **Return-to-stock errors**         | Scan every returned item back into inventory                  | Cashier / Floor Team |

## Cash Handling & Safe Procedures

### Cash Handling Rules

| Rule                            | Detail                                                                        |
| ------------------------------- | ----------------------------------------------------------------------------- |
| Dual custody for safe access    | Two keyholders present when safe is opened                                    |
| Drop schedule                   | Cash drops from register to safe every 2 hours or when drawer exceeds $500    |
| No personal items at register   | No phones, bags, or personal belongings within arm's reach of the cash drawer |
| Counterfeit detection           | UV light and counterfeit pen used on all $50 and $100 bills                   |
| Cash pick-up by armored carrier | Scheduled pickups; verified by Store Manager; logged with bag serial number   |

### Safe Reconciliation

| Task                              | Frequency               | Performed By                           |
| --------------------------------- | ----------------------- | -------------------------------------- |
| Count safe contents               | Opening and closing     | MOD + Cashier Lead                     |
| Reconcile deposits vs. POS totals | Daily                   | Store Manager                          |
| Investigate variances > $10       | As found                | Store Manager; escalate to LP if > $50 |
| Change order reconciliation       | Upon delivery from bank | Cashier Lead                           |

## Audit Program

### Audit Types

| Audit                        | Scope                                                        | Frequency             | Conducted By           |
| ---------------------------- | ------------------------------------------------------------ | --------------------- | ---------------------- |
| **Store LP audit**           | Full store review: cash, inventory, procedures, CCTV, EAS    | Quarterly             | LP Field Team          |
| **Cash audit**               | Register drawers, safe, deposits, change orders              | Monthly (unannounced) | LP or District Manager |
| **Receiving audit**          | ASN compliance, discrepancy logs, dock procedures            | Quarterly             | LP Field Team          |
| **EAS / tagging audit**      | Sample 100 units — verify EAS tag presence and functionality | Monthly               | Store LP Champion      |
| **CCTV functionality check** | All cameras operational, recording, properly aimed           | Weekly                | MOD                    |
| **Policy compliance audit**  | Associate adherence to bag check, discount, return policies  | Semi-annual           | LP + HR                |

### Audit Scoring

| Score    | Rating            | Action Required                                                                    |
| -------- | ----------------- | ---------------------------------------------------------------------------------- |
| 90–100   | Excellent         | Maintain current practices                                                         |
| 80–89    | Satisfactory      | Correct noted deficiencies within 30 days                                          |
| 70–79    | Needs Improvement | Corrective action plan within 14 days; re-audit in 60 days                         |
| Below 70 | Unsatisfactory    | Immediate corrective action; Store Manager performance review; re-audit in 30 days |

## Organized Retail Crime (ORC)

### ORC Indicators

| Indicator                                          | Action                                                            |
| -------------------------------------------------- | ----------------------------------------------------------------- |
| Repeated theft of same high-value items            | Report to LP; cross-reference with ORCA (ORC database)            |
| Groups entering and dispersing to high-theft zones | Notify MOD; increase zone coverage; activate CCTV live monitoring |
| Use of booster bags (foil-lined) to defeat EAS     | Report to LP immediately; do not confront                         |
| Fraudulent returns with no purchase history        | Flag in CFS; alert LP for pattern analysis                        |
| Recruitment of employees by external actors        | Anonymous tip line; LP investigation                              |

### ORC Response

- All suspected ORC activity reported to regional LP within 24 hours.
- LP coordinates with local law enforcement and industry coalitions (ORCA, RILA).
- Store-level associates do **not** confront suspected ORC groups; safety is paramount.

## CCTV & Technology

### CCTV Standards

| Requirement         | Specification                                                                                    |
| ------------------- | ------------------------------------------------------------------------------------------------ |
| Camera coverage     | 100% of entrances/exits, POS, fitting room exterior, receiving dock, safe area, high-theft zones |
| Recording retention | Minimum 30 days; 90 days for incident-related footage                                            |
| Resolution          | Minimum 1080p; facial identification quality at entry/exit                                       |
| Access              | LP, Store Manager, and District Manager only; access logged                                      |
| Signage             | "Premises under video surveillance" posted at all entrances per local law                        |

### Exception-Based Reporting (EBR)

The POS exception reporting system flags:

| Exception                          | Threshold                         |
| ---------------------------------- | --------------------------------- |
| Voids                              | > 3 per shift per associate       |
| No-sales                           | > 2 per shift per associate       |
| Post-voids                         | Any occurrence                    |
| Refunds (same associate)           | > 5 per day                       |
| High-value refunds                 | Single refund > $100              |
| Employee discount transactions     | Any — reviewed for compliance     |
| Sweet-hearting (items not scanned) | Detected via CCTV-POS integration |

## Training Requirements

### LP Training by Role

| Role               | Training Modules                                                                                           | Completion Deadline              |
| ------------------ | ---------------------------------------------------------------------------------------------------------- | -------------------------------- |
| **All associates** | Shoplifting awareness, customer service as deterrence, internal theft policy, bag check policy             | Within 14 days of hire           |
| **Cashiers**       | Cash handling, counterfeit detection, refund procedures, EBR awareness                                     | Within 14 days of hire           |
| **Shift Leads**    | All associate modules + apprehension protocol awareness, escalation procedures                             | Within 30 days of promotion      |
| **Store Managers** | All modules + investigation basics, audit readiness, ORC awareness                                         | Within 30 days of hire/promotion |
| **LP Officers**    | Full LP certification: apprehension, interview and interrogation, evidence handling, legal compliance, ORC | Pre-assignment                   |

### Annual Refresher

All store associates complete an annual LP refresher covering:

- Updated shrinkage results and trends
- Common theft methods (external and internal)
- Policy changes
- Case studies (anonymized)
