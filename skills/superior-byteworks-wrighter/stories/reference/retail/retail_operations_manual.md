## Opening Procedures

### Pre-Open Checklist (T-60 min to Store Open)

| #   | Task                                                            | Responsible         | Verified |
| --- | --------------------------------------------------------------- | ------------------- | -------- |
| 1   | Disarm security system, log entry code                          | Opening Manager     | ☐        |
| 2   | Walk perimeter — check locks, windows, signage                  | Opening Manager     | ☐        |
| 3   | Power on POS terminals, verify network connectivity             | Shift Lead          | ☐        |
| 4   | Count opening cash drawers against prior-night sealed envelopes | Cashier Lead        | ☐        |
| 5   | Inspect sales floor — reset displays, remove debris             | Floor Associates    | ☐        |
| 6   | Verify HVAC and lighting operational                            | Opening Manager     | ☐        |
| 7   | Review overnight shipment log, stage for processing             | Receiving Associate | ☐        |
| 8   | Check fitting rooms — clear, clean, hangers restocked           | Floor Associates    | ☐        |
| 9   | Confirm promotional signage matches current campaign            | Visual Merchandiser | ☐        |
| 10  | Unlock entrance doors at scheduled open time                    | Opening Manager     | ☐        |

### Cash Drawer Reconciliation

Each register begins the day with a standard float:

| Denomination    | Qty | Amount      |
| --------------- | --- | ----------- |
| $20 bills       | 5   | $100.00     |
| $10 bills       | 5   | $50.00      |
| $5 bills        | 10  | $50.00      |
| $1 bills        | 20  | $20.00      |
| Quarters        | 40  | $10.00      |
| Dimes           | 50  | $5.00       |
| Nickels         | 40  | $2.00       |
| Pennies         | 100 | $1.00       |
| **Total Float** |     | **$238.00** |

Variance exceeding **$2.00** from sealed envelope count must be reported to the Store Manager before opening.

## Closing Procedures

### End-of-Day Checklist (Store Close + 60 min)

| #   | Task                                                      | Responsible                    | Verified |
| --- | --------------------------------------------------------- | ------------------------------ | -------- |
| 1   | Final customer sweep — announce 15-min and 5-min warnings | Floor Associates               | ☐        |
| 2   | Lock entrance doors at scheduled close time               | Closing Manager                | ☐        |
| 3   | Run Z-reports on all POS terminals                        | Cashier Lead                   | ☐        |
| 4   | Count and reconcile each cash drawer                      | Cashier Lead                   | ☐        |
| 5   | Prepare bank deposit — dual-count, seal in tamper bag     | Closing Manager + Cashier Lead | ☐        |
| 6   | Clear fitting rooms, return merchandise to floor          | Floor Associates               | ☐        |
| 7   | Vacuum / sweep sales floor and stockroom                  | All Associates                 | ☐        |
| 8   | Secure stockroom door, verify locked                      | Closing Manager                | ☐        |
| 9   | Power down non-essential systems (displays, signage)      | Shift Lead                     | ☐        |
| 10  | Set alarm, exit via designated door, confirm armed status | Closing Manager                | ☐        |

### Deposit Protocol

- Deposits are placed in the safe for next-business-day armored pickup.
- If same-day night-drop is required, two associates must escort to the bank.
- Log deposit amount, bag number, and associate names in the **Daily Deposit Register**.

## Staffing Model

### Minimum Coverage Requirements

| Time Block  | Weekday              | Weekend              | Peak / Holiday       |
| ----------- | -------------------- | -------------------- | -------------------- |
| Open–12:00  | 3 associates + 1 MOD | 4 associates + 1 MOD | 5 associates + 2 MOD |
| 12:00–17:00 | 4 associates + 1 MOD | 5 associates + 1 MOD | 6 associates + 2 MOD |
| 17:00–Close | 3 associates + 1 MOD | 4 associates + 1 MOD | 5 associates + 2 MOD |

> **MOD** = Manager on Duty. At least one keyholder must be present at all times the store is open.

### Role Definitions

| Role                        | Core Responsibilities                                   |
| --------------------------- | ------------------------------------------------------- |
| **Store Manager (SM)**      | P&L ownership, hiring, training, district communication |
| **Assistant Manager (ASM)** | Scheduling, floor supervision, opening/closing          |
| **Shift Lead**              | Hourly team direction, POS overrides, escalation point  |
| **Sales Associate**         | Customer engagement, zone coverage, restocking          |
| **Cashier**                 | Transaction processing, loyalty sign-ups, bag checks    |
| **Receiving Associate**     | Shipment intake, ticketing, backstock organization      |
| **Visual Merchandiser**     | Planogram execution, window displays, signage           |

### Scheduling Guidelines

- Schedules published **two weeks** in advance.
- Shift-swap requests submitted via the scheduling platform no later than 48 hours before the shift.
- Overtime requires **prior written approval** from the Store Manager.
- No associate may work more than 6 consecutive days without a rest day.
- Minors (under 18) follow state/provincial labor-hour restrictions without exception.

## Daily Manager-on-Duty Responsibilities

| Time       | Activity                                                    |
| ---------- | ----------------------------------------------------------- |
| Store open | Conduct 5-minute stand-up huddle with opening team          |
| +1 hour    | Walk the floor — verify zone coverage, signage, cleanliness |
| Midday     | Review sales pace vs. daily plan; adjust labor if needed    |
| 15:00      | Check receiving — ensure all inbound processed              |
| 17:00      | Shift changeover briefing with evening team                 |
| Close      | Complete closing checklist, file daily recap report         |

### Daily Recap Report Contents

1. Net sales vs. plan and prior year
2. Transaction count and average transaction value (ATV)
3. Units per transaction (UPT)
4. Staffing actuals vs. schedule
5. Customer complaints or incidents
6. Inventory discrepancies noted
7. Facilities or maintenance issues

## Emergency Procedures

| Scenario               | Immediate Action                                                  | Report To                         |
| ---------------------- | ----------------------------------------------------------------- | --------------------------------- |
| Fire / fire alarm      | Evacuate via nearest exit; assemble at rally point                | 911, then District Manager        |
| Medical emergency      | Call 911; administer first aid if trained; do not move victim     | Store Manager, HR                 |
| Robbery / armed threat | Comply with demands; do not resist; activate silent alarm if safe | 911, then LP and District Manager |
| Power outage           | Suspend transactions; secure registers; guide customers to exit   | Facilities, District Manager      |
| Severe weather         | Move customers/associates to interior shelter area                | Monitor alerts; District Manager  |

> **Rally Point**: Designated meeting location (e.g., far corner of parking lot) documented and posted in the break room.

## Facilities & Maintenance

### Daily Maintenance Tasks

- Sweep and mop sales floor, fitting rooms, restrooms.
- Empty trash receptacles (sales floor, break room, restrooms, receiving).
- Wipe down POS counters and checkout belts.
- Clean glass doors and display cases.

### Scheduled Maintenance

| Task                         | Frequency     | Vendor / Internal  |
| ---------------------------- | ------------- | ------------------ |
| HVAC filter replacement      | Quarterly     | Vendor             |
| Deep carpet cleaning         | Semi-annually | Vendor             |
| Fire extinguisher inspection | Annually      | Vendor (certified) |
| Pest control treatment       | Monthly       | Vendor             |
| Parking lot sweeping         | Bi-weekly     | Vendor             |

Maintenance requests submitted via the **Facilities Ticket System** with priority classification:

| Priority      | Definition                          | Response SLA         |
| ------------- | ----------------------------------- | -------------------- |
| P1 — Critical | Safety hazard, store cannot operate | 4 hours              |
| P2 — Urgent   | Significant customer impact         | 24 hours             |
| P3 — Standard | Routine repair, no immediate impact | 5 business days      |
| P4 — Cosmetic | Minor aesthetic issue               | Next scheduled visit |
