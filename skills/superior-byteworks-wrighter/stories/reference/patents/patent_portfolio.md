### 3.4 Maintenance Fee Decision Framework

| Factor                | Criteria for Continued Payment             | Criteria for Lapse           |
| --------------------- | ------------------------------------------ | ---------------------------- |
| **Product Coverage**  | Patent covers current/planned products     | No product alignment         |
| **Licensing Revenue** | Patent generates or could generate revenue | No licensing potential       |
| **Defensive Value**   | Patent deters competitor activity          | No defensive purpose         |
| **Litigation Value**  | Patent has assertion potential             | No enforcement interest      |
| **Remaining Term**    | Significant term remaining                 | Near expiration              |
| **Maintenance Cost**  | Cost justified by value                    | Cost exceeds projected value |
