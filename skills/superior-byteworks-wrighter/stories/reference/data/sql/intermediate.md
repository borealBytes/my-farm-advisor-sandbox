# SQL Intermediate Schema Template

Use this for multi-entity systems with 4-8 tables and clear domain boundaries.

## Includes

- Mermaid ER with cardinality labels
- Normalized DDL scaffold (3NF)
- Foreign-key and lookup table pattern
