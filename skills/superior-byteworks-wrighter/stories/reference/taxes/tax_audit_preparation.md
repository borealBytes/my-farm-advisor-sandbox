## Audit Notification Review

### Initial Assessment

- [ ] Read audit notification letter in full
- [ ] Identify specific tax period(s) under examination
- [ ] Identify specific items or schedules under review
- [ ] Determine audit type (correspondence, office, field)
- [ ] Verify statute of limitations has not expired (IRC §6501)
- [ ] Note response deadline and mark on compliance calendar
- [ ] Confirm IRS agent credentials (badge number, office)
- [ ] Notify external tax advisor / tax attorney
- [ ] Notify CFO and legal counsel
- [ ] Evaluate need for Power of Attorney (Form 2848)

## Document Assembly Checklist

### Tax Returns and Workpapers

- [ ] Filed tax returns for audit period (federal and state)
- [ ] All schedules and attachments as filed
- [ ] Tax provision workpapers (ASC 740)
- [ ] Return-to-provision reconciliation
- [ ] Extension requests filed (Form 7004 / state equivalents)
- [ ] Estimated tax payment records and confirmation
- [ ] Prior year and subsequent year returns (for comparison)
- [ ] Amended returns filed during audit period (if any)

### Financial Records

- [ ] Audited or reviewed financial statements
- [ ] General ledger (detail and summary)
- [ ] Trial balance (pre-close and post-close)
- [ ] Chart of accounts with descriptions
- [ ] Journal entry detail with supporting documentation
- [ ] Consolidation elimination entries (if applicable)

### Revenue Documentation

- [ ] Revenue recognition policy documentation
- [ ] Sales journals and subledger detail
- [ ] Major customer contracts
- [ ] Deferred revenue schedules
- [ ] Bad debt write-off documentation and policy

### Expense Documentation

- [ ] Accounts payable subledger detail
- [ ] Vendor invoices for significant expenses
- [ ] Travel and entertainment documentation (IRC §274)
- [ ] Meals and entertainment log with business purpose
- [ ] Charitable contribution substantiation (IRC §170)
- [ ] Professional services invoices (legal, consulting, accounting)
- [ ] Rent and lease agreements

### Payroll Records

- [ ] Payroll registers for audit period
- [ ] Form 941 quarterly returns and reconciliation
- [ ] Form 940 annual FUTA return
- [ ] W-2/W-3 reconciliation to general ledger
- [ ] 1099-NEC/MISC for independent contractors
- [ ] Employee vs. contractor classification analysis
- [ ] Officer compensation documentation (IRC §162(m))
- [ ] Fringe benefit records (IRC §132)

### Asset Records

- [ ] Fixed asset register with additions and dispositions
- [ ] Depreciation schedules (book and tax)
- [ ] Section 179 / bonus depreciation elections
- [ ] Asset purchase agreements
- [ ] Asset sale documentation and gain/loss calculations

### Entity Records

- [ ] Articles of incorporation / operating agreement
- [ ] Board minutes and resolutions for audit period
- [ ] Organizational chart
- [ ] Related party transaction schedule
- [ ] Shareholder/partner ownership records

## Audit Day Protocols

### Before the Audit Meeting

- [ ] Designate a single point of contact for the auditor
- [ ] Prepare a private workspace for the auditor (if field audit)
- [ ] Brief all staff on audit protocols and communication rules
- [ ] Review prepared responses with external tax advisor
- [ ] Organize documents in indexed binders or secure digital folders

### During the Audit Meeting

- [ ] Welcome auditor professionally; verify credentials
- [ ] Provide only documents specifically requested
- [ ] Do NOT volunteer information beyond what is asked
- [ ] Take detailed notes on all questions asked and answers given
- [ ] Request clarification on any ambiguous questions before answering
- [ ] Consult with tax advisor before responding to complex questions
- [ ] Document all verbal requests and commitments
- [ ] Request written confirmation of any deadline extensions

### Communication Rules

| Rule                | Description                                                  |
| ------------------- | ------------------------------------------------------------ |
| Single spokesperson | All auditor communication goes through designated contact    |
| No speculation      | Never guess or speculate; say "I will need to research that" |
| Written records     | Confirm all verbal agreements in writing                     |
| Advisor present     | Tax advisor should attend all substantive meetings           |
| No originals        | Provide copies only; retain all originals                    |
| Scope control       | Push back on requests outside the stated audit scope         |

## Post-Audit Actions

### Upon Receiving Audit Findings

- [ ] Review examination report (Form 4549 / state equivalent) in detail
- [ ] Compare proposed adjustments to internal risk assessment
- [ ] Calculate total tax impact including interest and penalties
- [ ] Evaluate merits of each proposed adjustment
- [ ] Consult with tax advisor on appeal strategy

### Resolution Options

| Option                     | Timeline                          | Description                                   | Consideration                             |
| -------------------------- | --------------------------------- | --------------------------------------------- | ----------------------------------------- |
| Agree to findings          | 30 days from report               | Sign Form 870 / agreement                     | Fastest resolution; pay and close         |
| Partial agreement          | 30 days from report               | Agree to some items; contest others           | Negotiate on strongest positions          |
| Request Appeals conference | 30 days from report               | File formal protest per IRS Pub 5             | Independent review; no new evidence       |
| Tax Court petition         | 90 days from Notice of Deficiency | File petition in U.S. Tax Court               | Formal litigation; most expensive         |
| Pay and file claim         | 2 years from payment              | Pay tax; file refund claim (Form 1040X/1120X) | District Court or Court of Federal Claims |

### Process Improvement

- [ ] Document lessons learned from audit
- [ ] Update internal controls for identified weaknesses
- [ ] Revise record-keeping procedures as needed
- [ ] Update compliance calendar with new deadlines
- [ ] Train staff on areas where documentation was insufficient
- [ ] Schedule follow-up review in [X] months
