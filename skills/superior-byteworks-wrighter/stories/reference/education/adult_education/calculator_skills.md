# Calculator Skills for Adults

## Calculator Types

### Basic Calculator
- 4 operations (+, -, ×, ÷)
- Percent key
- Memory functions
- Square root

### Scientific Calculator
- All basic functions
- Trigonometry (sin, cos, tan)
- Logarithms (log, ln)
- Exponents and roots
- Statistics

### Graphing Calculator
- All scientific functions
- Graph equations
- Tables of values
- Programming capabilities

## Basic Operations

### Order of Operations

**PEMDAS on Calculator:**
1. **P**arentheses
2. **E**xponents
3. **M**ultiplication & **D**ivision (left to right)
4. **A**ddition & **S**ubtraction (left to right)

**Example:** 3 + 4 × 5
- Correct: 3 + (4 × 5) = 23
- Wrong: (3 + 4) × 5 = 35

**Practice:**
1. Calculate: 2 + 3 × 4 = {{ANSWER_1}}
2. Calculate: (2 + 3) × 4 = {{ANSWER_2}}
3. Calculate: 10 - 6 ÷ 2 = {{ANSWER_3}}

### Memory Functions

**Common Memory Keys:**
- **MC** - Memory Clear
- **MR** - Memory Recall
- **M+** - Memory Add
- **M-** - Memory Subtract
- **MS** - Memory Store

**Example:** Calculate total cost
```
Price 1: $45.67 [M+]
Price 2: $23.89 [M+]
Price 3: $67.45 [M+]
Tax 8%: [MR] × 0.08 = $10.88
Total: [MR] + tax = $147.89
```

## Scientific Calculator Functions

### Exponents and Roots

**Square:** x² or x^2
- Example: 5² = 25

**Cube:** x³ or x^3

**Square Root:** √ or √x
- Example: √25 = 5

**Cube Root:** ³√ or x^(1/3)

### Trigonometry

**Mode Setting:**
- **DEG** - Degrees
- **RAD** - Radians
- **GRAD** - Gradians

**Common Functions:**
- **sin** - Sine
- **cos** - Cosine
- **tan** - Tangent
- **sin⁻¹** - Arcsine
- **cos⁻¹** - Arccosine
- **tan⁻¹** - Arctangent

**Examples:**
- sin(30°) = 0.5
- cos(45°) = 0.7071
- tan(60°) = 1.732

**Right Triangle Problem:**
```
      /|
     / |
 10 /  | opposite
   /   |
  /____|
  angle = 30°

opposite = 10 × sin(30°) = 5
adjacent = 10 × cos(30°) = 8.66
```

### Logarithms

**Common Log:** log (base 10)
- log(100) = 2
- log(1000) = 3

**Natural Log:** ln (base e)
- ln(e) = 1
- ln(10) = 2.303

**Inverse:** 10^x or e^x
- 10^2 = 100
- e^1 = 2.718

## Workplace Calculations

### Finance

**Simple Interest:**
I = P × r × t
- P = Principal ($1000)
- r = Rate (0.05 for 5%)
- t = Time (3 years)
- I = 1000 × 0.05 × 3 = $150

**Compound Interest:**
A = P(1 + r)^t
- A = 1000(1 + 0.05)^3 = $1157.63

**Practice:**
- Loan: $5000 at 6% for 2 years = {{LOAN}}
- Investment: $2000 at 4% compounded annually for 5 years = {{INVEST}}

### Conversions

**Length:**
- Inches → cm: × 2.54
- Feet → meters: × 0.305
- Miles → km: × 1.609

**Weight:**
- Pounds → kg: × 0.454
- Ounces → grams: × 28.35

**Temperature:**
- °F to °C: (°F - 32) × 5/9
- °C to °F: (°C × 9/5) + 32

**Practice:**
- 68°F = {{TEMP_C}}°C

### Measurements

**Area:**
- Rectangle: length × width
- Circle: π × r²
- Triangle: ½ × base × height

**Volume:**
- Box: length × width × height
- Cylinder: π × r² × h

**Practice:**
- Room: 12' × 15' = {{AREA}} sq ft
- Paint needed (1 gal/400 sq ft): {{PAINT}} gallons

## GED Calculator (TI-30XS)

### Basic Setup

**Screen:**
- Shows current entry
- Displays answer
- Shows mode settings

**Key Features:**
- Toggle: [2nd] for alternate functions
- Clear: [CLEAR] or [2nd] [CLEAR] (quit)
- Mode: [MODE] to change settings

### Special Functions

**Fractions:**
Enter: [numerator] [⬆] [denominator]
Example: ¾ = [3] [⬆] [4]

**Mixed Numbers:**
Enter: [whole] [⬆] [numerator] [⬆] [denominator]
Example: 2½ = [2] [⬆] [1] [⬆] [2]

**Exponents:**
- x²: [x²]
- x³: [^] [3]
- Any power: [^]

**Roots:**
- √: [√]
- Any root: [⬆] [^]

**Scientific Notation:**
Enter: [number] [EE] [exponent]
Example: 3.5 × 10^6 = [3.5] [EE] [6]

### GED Test Tips

**Allowed:**
- Basic operations
- Scientific functions
- Fractions
- Exponents
- Roots

**Not Needed:**
- Graphing
- Programming
- Advanced stats

**Strategy:**
1. Learn calculator before test
2. Practice with same model
3. Use for complex calculations
4. Don't rely on it for basic math
