# GED Mathematical Reasoning Study Guide

## Test Overview

**Format:**
- Time: 115 minutes
- Questions: 46
- Calculator: Allowed on most (TI-30XS onscreen)
- Passing: 145+ (out of 200)

**Content:**
- Quantitative Problem Solving (45%)
- Algebraic Problem Solving (55%)

**Question Types:**
- Multiple choice
- Fill-in-the-blank
- Drag-and-drop
- Hot spot

## Calculator Skills

- Enter: Type number, press operation, type number, [=]

## Basic Operations

### Order of Operations (PEMDAS)

**Order:**
1. Parentheses
2. Exponents
3. Multiplication/Division (left to right)
4. Addition/Subtraction (left to right)

**Example:**
3 + 4 × 2³
= 3 + 4 × 8
= 3 + 32
= 35

**Practice:**
1. 2 + 3 × 4 = {{A1}}
2. (2 + 3) × 4 = {{A2}}
3. 10 - 6 ÷ 2 = {{A3}}
4. 2³ + 3² = {{A4}}

### Fractions

**Operations:**
- Add/Subtract: Common denominator
- Multiply: Top×top, bottom×bottom
- Divide: Multiply by reciprocal

**Example:**
½ + ⅓ = ³⁄₆ + ²⁄₆ = ⁵⁄₆
½ × ⅔ = ²⁄₆ = ⅓
½ ÷ ⅓ = ½ × ³⁄₂ = ¾

**Practice:**
1. ¾ + ½ = {{F1}}
2. ⅔ × ¾ = {{F2}}
3. ½ ÷ ¼ = {{F3}}

### Decimals

**Operations:**
- Add/Subtract: Line up decimals
- Multiply: Count decimal places
- Divide: Move decimal

**Converting:**
- Decimal to fraction: Place value
- Fraction to decimal: Divide

**Practice:**
1. 0.25 + 0.75 = {{D1}}
2. 0.4 × 0.5 = {{D2}}
3. ¾ = {{D3}} (decimal)

### Percentages

**Conversions:**
- % to decimal: Move decimal 2 left (50% = 0.50)
- Decimal to %: Move decimal 2 right (0.75 = 75%)

**Finding % of number:**
- 20% of 150 = 0.20 × 150 = 30

**Finding what %:**
- 30 is what % of 150? 30/150 = 0.20 = 20%

**Finding whole:**
- 30 is 20% of what? 30/0.20 = 150

**Practice:**
1. 15% of 200 = {{P1}}
2. 45 is what % of 180? {{P2}}
3. 60 is 30% of what? {{P3}}

**Metric System:**

- Mass: mg, g, kg
- Volume: mL, L
- Temperature: °C

**Conversions:**
- Kilo (k): 1000

**Practice:**

### Geometry

**Perimeter:**
- Rectangle: P = 2l + 2w
- Square: P = 4s
- Triangle: P = a + b + c
- Circle (circumference): C = 2πr

**Area:**
- Rectangle: A = lw
- Square: A = s²
- Triangle: A = ½bh
- Circle: A = πr²

**Volume:**
- Box: V = lwh
- Cylinder: V = πr²h

**Pythagorean Theorem:**
a² + b² = c² (right triangles)

**Practice:**

### Data Analysis

**Mean (Average):**
Sum of values ÷ Number of values

**Median:**
Middle value when ordered

**Mode:**
Most frequent value

**Range:**
Highest - lowest

**Practice:**

## Algebraic Problem Solving (55%)

### Expressions

**Variables:**
Letters representing unknown numbers

**Terms:**
Parts of expression separated by + or -

**Like Terms:**
Same variable part: 3x and 5x

**Simplifying:**
Combine like terms
3x + 2y + 4x - y = 7x + y

**Practice:**
1. Simplify: 2x + 3 + 4x - 1 = {{E1}}
2. Simplify: 3(a + 2) = {{E2}}

### Equations

**Solving One-Step:**
- x + 5 = 12 → x = 7
- 3x = 15 → x = 5
- x/4 = 3 → x = 12

**Solving Two-Step:**
2x + 5 = 13
2x = 8
x = 4

**Solving Multi-Step:**
3(x + 2) = 15
3x + 6 = 15
3x = 9
x = 3

**Practice:**
1. x - 7 = 3. x = {{X1}}
2. 4x = 28. x = {{X2}}
3. 2x + 6 = 14. x = {{X3}}
4. 3(x - 2) = 12. x = {{X4}}

### Inequalities

**Symbols:**
- <: Less than
- >: Greater than
- ≤: Less than or equal
- ≥: Greater than or equal

**Solving:**
Like equations, but flip sign when multiply/divide by negative

**Practice:**
1. x + 3 > 7. x {{I1}}
2. -2x ≤ 8. x {{I2}}

### Linear Functions

**Slope-Intercept Form:**
y = mx + b
m = slope, b = y-intercept

**Slope:**
m = (y₂ - y₁)/(x₂ - x₁)

**Finding from points:**
Given (2, 3) and (4, 7):
m = (7-3)/(4-2) = 4/2 = 2

**Practice:**
1. Slope of line through (1, 2) and (3, 8): {{S1}}
2. Equation with m=3, b=-2: y = {{EQ}}

### Systems of Equations

**Solving:**
- Substitution
- Elimination

**Example (Substitution):**
x + y = 7
x = y + 1

Substitute:
(y + 1) + y = 7
2y + 1 = 7
2y = 6
y = 3
x = 4

**Practice:**
1. Solve: x + y = 5, x = 2y. x = {{SYS1}}, y = {{SYS2}}

### Quadratic Equations

**Standard Form:**
ax² + bx + c = 0

**Solving Methods:**
- Factoring
- Quadratic formula

**Quadratic Formula:**
x = (-b ± √(b² - 4ac)) / 2a

**Practice:**
1. Solve: x² - 5x + 6 = 0. x = {{Q1}}
2. Using formula: x² + 4x + 3 = 0. x = {{Q2}}

## Word Problems

### Strategy

2. **Identify what you know**

### Types

**Money:**
Total = (quantity × price)

**Mixtures:**
(amount₁ × %) + (amount₂ × %) = total %

**Practice:**

2. Tickets cost $10 for adults, $6 for children. 5 adults and 3 children = ${{WP2}}

- Memorize formulas
- Practice word problems

- Sign errors
- Dropping negative
- Wrong formula
- Calculator entry error
- Not checking
