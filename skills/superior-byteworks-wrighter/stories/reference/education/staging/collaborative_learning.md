## Role Descriptions

### Facilitator
- Keeps group on task
- Manages time
- Ensures everyone participates

### Recorder
- Takes notes
- Writes final product
- Documents decisions

### Materials Manager
- Gets supplies
- Returns items
- Organizes workspace

### Reporter
- Shares with class
- Answers questions
- Presents findings

## Group Contract

We agree to:
- [ ] Listen to each other
- [ ] Share ideas respectfully
- [ ] Complete our assigned parts
- [ ] Ask for help when needed
- [ ] Help others when asked
