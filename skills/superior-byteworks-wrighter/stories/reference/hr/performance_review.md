## Rating Scale

All performance dimensions are evaluated using the following standardized scale:

| Rating                   | Score | Definition                                                      |
| ------------------------ | ----- | --------------------------------------------------------------- |
| **Exceptional**          | 5     | Consistently exceeds all expectations; serves as a role model   |
| **Exceeds Expectations** | 4     | Frequently exceeds expectations in most areas of responsibility |
| **Meets Expectations**   | 3     | Consistently fulfills all requirements of the position          |
| **Needs Improvement**    | 2     | Performance falls below expectations in one or more key areas   |
| **Unsatisfactory**       | 1     | Performance is consistently below minimum acceptable standards  |
