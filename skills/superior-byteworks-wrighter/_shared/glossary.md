# Glossary

Terms used across wrighter domains.

## Core Terms

| Term | Definition |
|------|------------|
| **Domain** | Subdomain (stories, visuals, etc.) |
| **Tier** | Complexity: 1=router, 2=light, 3=full |
| **Router** | Entry point linking to resources |
| **Template** | Reusable document structure |
| **FAS** | Full Authoritative Skill format |
