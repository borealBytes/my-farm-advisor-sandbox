# Attribution & Creation Notes

## Original Creation Method

This artwork was created using a **greenfield approach** - meaning every
element was built from scratch without reference to official artwork.

### Design Process

**1. Visual Language Study (from memory/observation)**
- Yellow skin tone (#FDD015) - distinctive to The Simpsons
- Spiky hair for Bart - iconic silhouette
- Two hairs for Homer - recognizable trait
- Cylindrical heads - show style
- Four-fingered hands (implied, not shown)

**2. Geometric Abstraction**
```
Bart's head = cylinder + spikes
- Cylinder: <path> with rounded top
- Spikes: <path> with zigzag points
- No reference images used
```

**3. From Scratch Implementation**
```svg
<!-- NOT traced from image -->
<path d="M -45 0..." />

<!-- Hand-coded coordinates -->
<circle cx="-18" cy="25" r="15" />
```

### Why It Looks Authentic

**Color Palette:**
- Yellow skin: #FDD015 (publicly known)
- Blue pants: #5C94FC (iconic color)
- Red shirt: #E84A24 (Bart's color)
- These are color choices, not copied artwork

**Proportions:**
- Oversized heads
- Skinny bodies
- Four fingers (implied)
- These are stylistic choices

**Expressions:**
- "Don't have a cow" smirk
- "D'oh" shock face
- These are character mannerisms

### Technical Differences from Official

**Official Simpsons Animation:**
- Uses proprietary software
- Cel-shaded animation
- Vector but closed format
- Precise animation rigs

**This SVG:**

- Simplified geometry
- Single static pose
- Educational purpose
