# User Journey — Simple (3–6 tasks)

Single section journey. Use for basic user flows and quick experience mapping.

---

## Example: App Onboarding

```mermaid
journey
 accTitle: App Onboarding Journey
 accDescr: Simple user journey showing onboarding experience with satisfaction scores

 section Onboarding
  Download app: 4: User
  Create account: 3: User
  Verify email: 2: User
  Complete profile: 3: User
  First use: 5: User
```

---

## Example: Purchase Flow

journey

## Copy-Paste Template

```mermaid
journey
 accTitle: REPLACE WITH 3-8 WORD TITLE
 accDescr: REPLACE WITH 1-2 sentences describing what this journey shows

 section Journey Phase
  Task 1: 4: Actor
  Task 2: 3: Actor
  Task 3: 5: Actor
  Task 4: 2: Actor
```

---

## Tips

- Single section works for simple journeys
- Use scores to highlight pain points (low) and delights (high)
- 3–6 tasks covers most simple flows
- Focus on the primary actor (usually the user)
