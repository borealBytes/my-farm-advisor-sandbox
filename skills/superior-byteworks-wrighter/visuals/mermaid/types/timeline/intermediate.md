# Timeline — Intermediate (6–12 events)

Multi-section timeline. Use for roadmaps with parallel tracks or grouped events.

---

## Example: Quarterly Roadmap

```mermaid
timeline
 accTitle: Quarterly Product Roadmap
 accDescr: Multi-quarter roadmap showing features and milestones across Q1-Q2

 section Q1 2024
 Auth : OAuth integration
 Dashboard : Analytics v1
 API : REST endpoints

 section Q2 2024
 Mobile : iOS app beta
 Export : PDF reports
 Search : Full-text search
```

---

```mermaid
timeline
 accTitle: REPLACE WITH 3-8 WORD TITLE
 accDescr: REPLACE WITH 1-2 sentences describing what this timeline shows

 section Phase A
 Event 1 : Description
 Event 2 : Description
 Event 3 : Description

 section Phase B
 Event 4 : Description
 Event 5 : Description
 Event 6 : Description
```

---

## Tips

- Use sections to group by quarter, team, or phase
- 2–4 events per section is ideal
- Keep section names short (1–2 words)
- Parallel sections help compare tracks
