# WRIGHTER Visuals

Text-based visual documentation for Omni.

| Format      | Router               | For...                              |
| ----------- | -------------------- | ----------------------------------- |
| **Mermaid** | [mermaid/](mermaid/) | Diagrams, flows, charts (preferred) |
| **SVG**     | [svg/](svg/)         | Custom illustrations, characters    |
