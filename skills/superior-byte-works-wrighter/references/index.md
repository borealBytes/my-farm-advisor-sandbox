## ## Statistical Reporting

| Resource                            | Description                              |
| ----------------------------------- | ---------------------------------------- |
| EQUATOR Network                     | Reporting guidelines for health research |
| CONSORT extension for N-of-1 trials | Single-subject designs                   |
| GRADE handbook                      | Evidence certainty assessment            |
| Cochrane Handbook                   | Systematic review methods                |

**Key statistical reporting rules:**

- Report effect sizes with 95% confidence intervals, not just p-values
- Use two-sided p-values unless one-sided is pre-specified and justified
- Report exact p-values (e.g., p = 0.03) not inequalities (p < 0.05) when p > 0.001
- For p < 0.001, report as "p < 0.001"
