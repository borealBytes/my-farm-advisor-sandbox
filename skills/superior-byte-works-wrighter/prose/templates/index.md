## ## Common Prose Patterns

### The Claim-Evidence-Implication Paragraph

```markdown
[Claim: state the point directly.]
[Evidence: data, citation, example, or derivation.]
[Implication: what follows from the evidence.]
```

### The Problem-Solution-Trade-off Pattern

```markdown
**Problem:** [What is broken or missing?]
**Solution:** [What we propose.]
**Trade-offs:** [What we give up; when this is not the right choice.]
```

### The Before-After Pattern

```markdown
**Before:** [Current state — specific, concrete]
**After:** [Desired state — specific, concrete]
**How:** [The change that gets us there]
```

### The STAR Pattern (for case studies)

```markdown
**Situation:** [Context and background]
**Task:** [What needed to be done]
**Action:** [What was done and why]
**Result:** [Measurable outcome]
```
