# High School Preparation Checklist

## Academic Readiness

- [ ] Understand GPA system
- [ ] Know graduation requirements
- [ ] Selected 9th grade courses
- [ ] Understand credit system
- [ ] Know how to calculate GPA

## Organizational Skills

- [ ] Use a planner consistently
- [ ] Organize materials by subject
- [ ] Track assignments digitally
- [ ] Manage long-term projects
- [ ] Balance academics and activities

## Self-Advocacy

- [ ] Know when to ask for help
- [ ] Can explain my learning needs
- [ ] Comfortable approaching teachers
- [ ] Know how to access resources
- [ ] Can self-assess my work

## Social-Emotional

- [ ] Identified potential clubs/activities
- [ ] Plan to attend orientation
- [ ] Know how to navigate new school
- [ ] Have strategies for stress
- [ ] Set personal goals for 9th grade

## Summer Preparation

- [ ] Summer reading completed
- [ ] Math skills reviewed
- [ ] School supplies purchased
- [ ] Schedule received
- [ ] Transportation plan made
