### Using Folders

**What is a Folder?**
- Like a filing cabinet drawer
- Holds files
- Keeps organized

**Creating Folders:**
1. Right-click empty space
2. Select "New" → "Folder"
3. Type name
4. Press Enter

**Your Folder System:**

```
Documents/
├── {{FOLDER_1}}/
│   ├── {{FILE_1_1}}
│   └── {{FILE_1_2}}
├── {{FOLDER_2}}/
│   └── {{FILE_2_1}}
├── {{FOLDER_3}}/
└── {{FOLDER_4}}/
```

**Organizing Tips:**
- One folder per project/course
- Name clearly
- Use dates if helpful
- {{ORGANIZATION_TIPS}}

### File Explorer (or Finder)

**Opening:**
- Click folder icon
- Or Start → File Explorer

**Navigation:**
- Back arrow: go back
- Address bar: shows location
- Folders: double-click to open
- Up arrow: go up one level

**Views:**
- Icons
- List
- Details
- {{PREFERRED_VIEW}}

### File Operations

**Creating New File:**
1. Right-click
2. New → File type
3. Name it
4. Press Enter

**Copying:**
1. Select file
2. Right-click → Copy
3. Go to new location
4. Right-click → Paste

**Moving:**
1. Drag to new location
- OR -
1. Cut (Ctrl+X)
2. Go to new location
3. Paste (Ctrl+V)

**Deleting:**
1. Select file
2. Press Delete key
3. Confirm
4. (Goes to Recycle Bin/Trash)

**Renaming:**
1. Right-click file
2. Rename
3. Type new name
4. Press Enter

**Shortcuts:**
- Copy: Ctrl+C
- Paste: Ctrl+V
- Cut: Ctrl+X
- Select All: Ctrl+A
- Undo: Ctrl+Z
