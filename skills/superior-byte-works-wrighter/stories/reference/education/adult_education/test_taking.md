## Test Format Strategies

### Multiple Choice

**The Systematic Approach:**

**1. Read the Question First:**
- Don't look at answers yet
- Predict the answer
- {{PREDICTION_1}}

**2. Read All Options:**
- Don't stop at first right-sounding answer
- Eliminate wrong answers
- {{ELIMINATE_1}}

**3. Eliminate Obviously Wrong:**
- Cross out wrong answers
- Narrow down choices
- {{NARROWED_1}}

**4. Compare Remaining:**
- Which is most correct?
- Look for qualifiers (always, never, sometimes)
- {{COMPARED_1}}

**5. Check Your Answer:**
- Reread question with your answer
- Does it make sense?
- {{CHECKED_1}}

**Strategies:**

**Process of Elimination:**
- A. {{CHOICE_A}} - Eliminate: {{REASON_A}}
- B. {{CHOICE_B}} - Eliminate: {{REASON_B}}
- C. {{CHOICE_C}} - Keep: {{REASON_C}}
- D. {{CHOICE_D}} - Keep: {{REASON_D}}

**Look for:**
- ✅ Absolute words (always, never, all, none) - often wrong
- ✅ Qualifiers (usually, often, may) - often right
- ✅ Similar answers - one is usually correct
- ✅ Longest answer - often correct
- ✅ "All of the above" - if two others seem right, likely this

**When Unsure:**
- Don't leave blank (unless penalty for wrong)
- Make educated guess
- Eliminate first
- Go with first instinct (usually right)

### True/False

**Tips:**
- Watch for qualifiers (sometimes, usually, rarely)
- Absolute words (always, never, all) often make false
- Every part must be true for whole to be true
- Read carefully

**Examples:**
- "All mammals lay eggs" → False (most don't)
- "Most mammals give live birth" → True

### Matching

**Strategy:**
1. Read all items first
2. Match ones you know for sure
3. Cross out used answers
4. Use process of elimination for rest

### Short Answer

**Strategy:**
- Answer directly
- Be specific
- Use examples
- Complete sentences
- Check spelling

## Answer Strategies

### When You Don't Know

**Multiple Choice:**
1. Eliminate obviously wrong
2. Look for patterns
3. Choose best guess
4. Move on

**Essay:**
1. Write something
2. Use what you know
3. Connect to topic
4. Show reasoning

**Never Leave Blank:**
- No penalty for guessing on most tests
- Partial credit possible
- Something > nothing

### Managing Mistakes

**If You Realize Mistake:**
- Fix it if you have time
- Be sure you're right
- Don't second-guess too much
- Trust first instinct (usually correct)

**If Unsure:**
- Stick with first answer (usually right)
- Only change if you find proof
- {{CHANGE_DECISION}}

## Different Test Types

### Standardized Tests (GED, SAT, etc.)

**General Strategy:**
- Pace yourself
- Skip hard questions
- Use process of elimination
- Guess if unsure (no penalty)
- Manage time

**Math Section:**
- Show work on scratch paper
- Check calculations
- Estimate first
- Use calculator wisely
- {{MATH_STRATEGY}}

**Reading Section:**
- Read questions first
- Skim passage
- Look for keywords
- Refer back to text
- {{READING_STRATEGY}}

**Writing Section:**
- Read carefully
- Look for grammar errors
- Check sentence structure
- Trust your ear
- {{WRITING_STRATEGY}}

### Online Tests

**Before:**
- Test technology
- Check internet
- Have backup plan
- Find quiet space

**During:**
- Stay focused
- Don't open other tabs
- Save regularly (if allowed)
- Check time

### Open-Book Tests

**Don't:**
- Don't rely on looking everything up
- Don't waste time searching
- Don't copy directly

**Do:**
- Prepare like closed-book
- Know where to find info
- Use for verification
- Focus on understanding

### Timed Tests

**Strategy:**
- Work quickly but carefully
- Skip hard questions
- Return if time
- Don't rush basics
- Watch clock

## Test Anxiety

### Before Test

**Manage Stress:**
- Get good sleep
- Eat well
- Exercise
- Breathe deeply
- Positive self-talk

**Positive Affirmations:**
- "I am prepared"
- "I can do this"
- "I know this material"
- "I will do my best"
- {{YOUR_AFFIRMATION}}

### During Test

**If Anxious:**
- Take deep breaths
- Stretch
- Positive self-talk
- Focus on one question
- Skip if needed

**Quick Calm:**
- Inhale 4 counts
- Hold 4 counts
- Exhale 4 counts
- Repeat 3 times

**Grounding Technique:**
- 5 things you see
- 4 things you touch
- 3 things you hear
- 2 things you smell
- 1 thing you taste
