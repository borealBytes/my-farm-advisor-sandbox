# Note-Taking Strategies

## Why Note-Taking Matters

**Benefits:**
- Improves focus and attention
- Organizes information
- Creates study material
- Enhances memory
- Tracks progress

**Research Shows:**
- Taking notes increases retention by 20-30%
- Reviewing notes within 24 hours improves recall
- Writing by hand engages brain differently than typing

## The Cornell Method

### Setup

Divide page into three sections:
```
| Cue Column    | Notes Column          |
| (2.5")        | (6")                  |
|               | Main notes here       |
| Keywords      | - Bullet points       |
| Questions     | - Details             |
| Main ideas    | - Examples            |
|               |                       |
|_______________|_______________________|
| Summary (2")                          |
| Brief summary of entire page          |
```

### How to Use

**During Class/Reading:**
1. Write notes in right column
2. Use abbreviations
3. Skip lines between ideas
4. Mark unclear points (?)

**After Class:**
1. Review notes within 24 hours
2. Fill in gaps
3. Write questions in left column
4. Summarize bottom section

**Study:**
1. Cover notes column
2. Read questions/keywords
3. Recite answers out loud
4. Check accuracy

## Mind Mapping

### What is Mind Mapping?
Visual way to organize ideas around a central concept.

### How to Create

**Step 1: Central Topic**
- Write main topic in center
- Circle it
- Use image if helpful

**Step 2: Main Branches**
- Draw lines from center
- Label with key themes
- Use different colors

**Step 3: Sub-branches**
- Add details to branches
- Connect related ideas
- Use keywords, not sentences

**Step 4: Add Details**
- Examples
- Facts
- Questions
- Connections

### Example

```
                    [Central Topic]
                           |
          _________________|_________________
         |                  |                |
    [Branch 1]        [Branch 2]       [Branch 3]
         |                  |                |
    [Sub 1]            [Sub 4]          [Sub 7]
    [Sub 2]            [Sub 5]          [Sub 8]
    [Sub 3]            [Sub 6]          [Sub 9]
```

## The Outline Method

### Structure

Use hierarchy with indentation:

**I. Main Topic**

      1. Detail
      2. Detail
   B. Sub-topic
      1. Detail
      2. Detail

**II. Main Topic**
   A. Sub-topic
   B. Sub-topic

## Active Listening Strategies

### Before Taking Notes

**Prepare:**
- [ ] Review previous notes
- [ ] Read assigned material
- [ ] Bring materials
- [ ] Arrive early
- [ ] Sit where you can see/hear

### During Note-Taking

**Listen First:**
- Don't write everything
- Focus on main ideas
- Listen for emphasis
- Watch for signals

**Key Signals:**
- "This is important..."
- "Remember..."
- "The main point is..."
- Repetition
- Writing on board
- Tone change

**Write Later:**
- Jot keywords
- Listen to example
- Then write explanation

### After Note-Taking

**Within 24 Hours:**
- [ ] Review notes
- [ ] Fill in gaps
- [ ] Clarify abbreviations
- [ ] Add questions
- [ ] Highlight key points
- [ ] Create summary

**Weekly:**
- [ ] Review all notes
- [ ] Connect related topics
- [ ] Create study guide
- [ ] Test yourself

## Review Strategies

### The 24-Hour Rule

**Within 1 day:**
- Review notes
- Fill gaps
- Clarify
- Organize

**Benefit:**
- Memory still fresh
- Fill while you remember
- Prevents forgetting

### Weekly Review

**Every Week:**
- [ ] Review all week's notes
- [ ] Create summary sheet
- [ ] Connect related topics
- [ ] Update study guide
- [ ] Plan next week

### Before Tests

**Study Process:**
1. Review notes (not textbook)
2. Create practice questions
3. Teach someone else
4. Self-test
5. Review weak areas

## Common Mistakes

**Avoid These:**
- ❌ Writing everything word for word
- ❌ Not reviewing notes
- ❌ Disorganized notes
- ❌ Missing key points
- ❌ Not asking questions
- ❌ Poor handwriting
- ❌ No dates/labels

**Fix Them:**
- ✅ Summarize in own words
- ✅ Review within 24 hours
- ✅ Use consistent format
- ✅ Listen for main ideas
- ✅ Ask for clarification
- ✅ Write clearly
- ✅ Label everything
