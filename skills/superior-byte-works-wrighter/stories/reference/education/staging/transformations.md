# Geometric Transformations

## Types of Transformations

| Type | Description | Rule | Preserves |
|------|-------------|------|-----------|
| Translation | Slide | $(x, y) \to (x + a, y + b)$ | Size, shape |
| Reflection | Flip | Across line | Size, shape |
| Rotation | Turn | $(x, y) \to (-y, x)$ for 90° | Size, shape |
| Dilation | Resize | $(x, y) \to (kx, ky)$ | Shape |
