# Pythagorean Theorem Applications

## The Theorem

$a^2 + b^2 = c^2$

Where:
- $a$ = {{LEG_A}}
- $b$ = {{LEG_B}}
- $c$ = hypotenuse (longest side)

## Finding the Hypotenuse

Triangle with legs {{LEG_1}} and {{LEG_2}}:

$a^2 + b^2 = c^2$

${{LEG_1}}^2 + {{LEG_2}}^2 = c^2$

${{SQUARED_1}} + {{SQUARED_2}} = c^2$

$c^2 = {{SUM}}$

$c = \sqrt{{SUM}} = {{ANSWER}}$

## Finding a Leg

Hypotenuse = {{HYP}}, one leg = {{KNOWN_LEG}}

$c^2 - a^2 = b^2$

${{HYP}}^2 - {{KNOWN_LEG}}^2 = b^2$

$b = {{MISSING_LEG}}$

## Distance on Coordinate Plane

Point A: ({{X1}}, {{Y1}})
Point B: ({{X2}}, {{Y2}})

Distance = $\sqrt{({{X2}} - {{X1}})^2 + ({{Y2}} - {{Y1}})^2}$

= $\sqrt{{DX}}^2 + {{DY}}^2}$

= $\sqrt{{SUM_SQUARES}}$

= {{DISTANCE}}
