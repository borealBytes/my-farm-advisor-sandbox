# Equations with Exponents

## Laws of Exponents

| Law | Rule | Example |
|-----|------|---------|
| Product | $x^m \cdot x^n = x^{m+n}$ | {{PROD_EX}} |
| Quotient | $\frac{x^m}{x^n} = x^{m-n}$ | {{QUOT_EX}} |
| Power | $(x^m)^n = x^{mn}$ | {{POW_EX}} |
| Zero | $x^0 = 1$ | {{ZERO_EX}} |
| Negative | $x^{-n} = \frac{1}{x^n}$ | {{NEG_EX}} |

## Solving Exponential Equations

### Example 1

$2^x = 32$

$2^x = 2^5$

Therefore: $x = {{ANS_1}}$

### Example 2

$5^{x+1} = 125$

$5^{x+1} = 5^3$

$x + 1 = 3$

$x = {{ANS_2}}$

### Example 3

$3^{2x-1} = 27$

$3^{2x-1} = 3^3$

$2x - 1 = 3$

$2x = 4$

$x = {{ANS_3}}$
