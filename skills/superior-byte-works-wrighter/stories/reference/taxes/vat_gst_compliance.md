## Compliance Checklist

### Pre-Filing

- [ ] All sales invoices issued with correct VAT/GST treatment
- [ ] All purchase invoices received and VAT/GST codes verified
- [ ] Reverse charge entries recorded for cross-border services
- [ ] Credit notes processed and reflected in return
- [ ] Bad debt relief claims prepared (where applicable)
- [ ] Partial exemption calculation updated
- [ ] Intercompany transactions reviewed for correct VAT treatment
- [ ] Exchange rates applied per local rules (ECB, HMRC, etc.)

### Filing

- [ ] Return completed in local filing system (MTD, SII, SAF-T, etc.)
- [ ] Amounts reconciled to accounting system
- [ ] Return reviewed and approved by Tax Manager
- [ ] Return submitted by deadline
- [ ] Payment remitted by deadline
- [ ] Filing confirmation saved to records

### Post-Filing

- [ ] Confirmation receipt archived
- [ ] Payment confirmation recorded
- [ ] Variances from prior period analyzed
- [ ] Refund claim tracked (if applicable)
- [ ] Supporting documentation filed per retention policy
