## Supporting Documentation Checklist

### Financial Records

- [ ] Audited or reviewed financial statements (GAAP basis)
- [ ] General ledger fully reconciled for tax year
- [ ] Trial balance (pre-close and post-close)
- [ ] Consolidation workpapers (if applicable)
- [ ] Intercompany transaction schedules
- [ ] Journal entry detail with supporting documentation

### Bank and Cash

- [ ] Bank statements for all accounts (12 months)
- [ ] Bank reconciliations for all accounts (12 months)
- [ ] Investment account statements
- [ ] Petty cash reconciliation

### Revenue and Receivables

- [ ] Revenue recognition schedules per ASC 606
- [ ] Accounts receivable aging report
- [ ] Bad debt write-off documentation
- [ ] Deferred revenue schedule

### Expenses and Payables

- [ ] Accounts payable aging report
- [ ] Accrual schedules with supporting detail
- [ ] Prepaid expense amortization schedules
- [ ] Travel and entertainment log (IRC §274 compliance)
- [ ] Charitable contribution receipts (IRC §170)

### Payroll

- [ ] W-2 reconciliation to general ledger
- [ ] Form 941 reconciliation to payroll records
- [ ] Officer compensation analysis (IRC §162(m))
- [ ] Fringe benefit valuation (IRC §132)
- [ ] Retirement plan contribution records

### Fixed Assets

- [ ] Fixed asset register (additions, dispositions, transfers)
- [ ] Depreciation schedules (book vs. tax per IRC §167/168)
- [ ] Section 179 election documentation (IRC §179)
- [ ] Bonus depreciation tracking (IRC §168(k))
- [ ] Asset impairment analysis

### Entity and Ownership

- [ ] Articles of incorporation / operating agreement (current)
- [ ] Shareholder/partner equity roll-forward
- [ ] Distribution / dividend records
- [ ] Stock option / equity compensation records
- [ ] Related party transaction disclosures

## Tax Provision Workpapers (ASC 740)

- [ ] Current tax provision calculation (federal and state)
- [ ] Deferred tax asset/liability roll-forward
- [ ] Effective tax rate reconciliation
- [ ] Uncertain tax position analysis (FIN 48 / ASC 740-10)
- [ ] Valuation allowance assessment
- [ ] Return-to-provision adjustment from prior year
- [ ] Tax rate change impact analysis
- [ ] Intraperiod tax allocation
