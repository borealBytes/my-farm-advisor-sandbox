## Voice & Tone

### Brand Voice

Our voice is the consistent personality behind all communications. It does not change.

| Voice Attribute | Definition                         | Example           |
| --------------- | ---------------------------------- | ----------------- |
| [Attribute 1]   | [e.g., Confident but not arrogant] | [Sample sentence] |
| [Attribute 2]   | [e.g., Clear, not oversimplified]  | [Sample sentence] |
| [Attribute 3]   | [e.g., Warm, not unprofessional]   | [Sample sentence] |

### Tone Spectrum

Tone adjusts based on context while voice remains constant.

| Context            | Tone                    | Example                                 |
| ------------------ | ----------------------- | --------------------------------------- |
| Marketing / Ads    | Energetic, aspirational | "Transform the way you work."           |
| Product UI         | Clear, helpful          | "Your changes have been saved."         |
| Support / Help     | Empathetic, direct      | "We're sorry for the inconvenience."    |
| Legal / Compliance | Precise, neutral        | "Your data is processed in accordance." |
| Internal / Culture | Casual, inclusive       | "Welcome to the team!"                  |

### Writing Guidelines

- Use active voice; avoid passive constructions
- Prefer short sentences (15-20 words average)
- Avoid jargon unless writing for technical audiences
- Use second person ("you") for customer-facing copy
- Capitalize product names; do not capitalize generic terms
- Oxford comma: [Yes / No]
- Date format: [DD-MMM-YYYY / Month DD, YYYY]
