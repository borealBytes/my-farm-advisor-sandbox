# Mindmap — Simple (5–10 nodes)

Single concept with 2–3 branches. Use for quick brainstorming and simple categorization.

---

## Example: Project Planning

```mermaid
mindmap
 accTitle: Project Planning Mindmap
 accDescr: Simple mindmap showing key project planning categories and initial tasks

 root((📋 Project))
  🎯 Goals
   Define scope
   Set milestones
  👥 Team
   Assign roles
   Schedule kickoff
  ⚙️ Tools
   Setup repo
   Configure CI
```

---

## Tips

- 2–3 first-level branches is ideal for simple maps
- Keep to 2 levels deep (root + children + grandchildren)
- Use emojis on root and first-level nodes
- Keep leaf nodes to 1–3 words
