# Timeline — Simple (3–6 events)

Single section timeline. Use for basic chronologies and quick event sequences.

---

## Example: Project Milestones

```mermaid
timeline
 accTitle: Project Milestone Timeline
 accDescr: Simple timeline showing key project milestones from kickoff to launch

 section Project
 Kickoff : Team assembled
 Design : Mockups approved
 Development : Core features built
 Testing : QA complete
 Launch : 🚀 v1.0 released
```

---

timeline

## Copy-Paste Template

```mermaid
timeline
 accTitle: REPLACE WITH 3-8 WORD TITLE
 accDescr: REPLACE WITH 1-2 sentences describing what this timeline shows

 section Phase Name
 Event 1 : Description
 Event 2 : Description
 Event 3 : Description
 Event 4 : Description
```

---

## Tips

- Single section is ideal for simple timelines
- 3–6 events provides good coverage without clutter
- Use emojis on milestone events
- Keep descriptions to 2–4 words
