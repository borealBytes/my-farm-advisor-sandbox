# Kanban — Simple (3–6 cards)

Basic task board. Use for simple task tracking.

---

## Example: Sprint Tasks

```mermaid
kanban
 accTitle: Sprint Task Board
 accDescr: Simple kanban board showing sprint tasks by status

 Todo
  [Setup project]
  [Configure CI]

 In Progress
  [Implement auth]

 Done
  [Create repo]
  [Setup team]
```

---

## Copy-Paste Template

```mermaid
kanban
 accTitle: REPLACE WITH 3-8 WORD TITLE
 accDescr: REPLACE WITH 1-2 sentences describing what this board shows

 Todo
  [Task 1]
  [Task 2]

 In Progress
  [Task 3]

 Done
  [Task 4]
```

---

## Tips

- 3–6 cards is ideal for simple boards
- Use standard columns (Todo, In Progress, Done)
- Keep task names concise
- Consider adding IDs for tracking
